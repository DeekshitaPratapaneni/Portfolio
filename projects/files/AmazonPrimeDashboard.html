Parking Space Detection: Using Computer Vision

Problem Statement: This project aims to develop a smart parking system that detects vacant and occupied spaces in real-time from CCTV footage using Computer Vision

Actions Taken: The parking lot footage was processed using OpenCV and deep learning-based object detection models. A pre-trained CNN model was fine-tuned to detect parked vehicles, and a segmentation mask was applied to mark parking slots. The system used frame-by-frame analysis to dynamically update parking availability. Optimization techniques such as image preprocessing, adaptive thresholding, and morphological operations were applied to improve detection accuracy

Results: The smart parking detection system achieved high accuracy, correctly identifying occupied and vacant spaces in real-time. The model effectively handled varying lighting conditions and occlusions. Implementing this system can reduce parking search time, lower emissions, and improve traffic flow in urban areas. The solution is scalable and can be integrated with IoT-based smart city infrastructure for enhanced efficiency
